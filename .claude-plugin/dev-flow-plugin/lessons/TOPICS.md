# dev-flow Plugin Lessons — Topic Index

This directory contains cross-project lessons surfaced to agents during implementation.

## Index

Lessons are organized by category. Each lesson file includes `framework`, `phase`,
`severity`, and `stack` frontmatter for context-aware filtering.

---

## architecture/

Lessons about component patterns, layer boundaries, and ports & adapters.

## framework-nuxt/

Lessons specific to Nuxt.js / Nuxt layers development.

## security/

Lessons about security vulnerabilities, injection risks, and auth patterns.

## testing/

Lessons about test runners, E2E tests, and test isolation.

## workflow/

Lessons about process gaps, phase gaps, and gate failures.
